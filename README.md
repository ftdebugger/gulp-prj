Prepare
=======

    ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"
    brew install node
    npm install -g gulp
    npm install -g bower

Install
=======

    git clone git@github.com:dontuncleme/gulp-prj.git MY_PATH && cd MY_PATH
    rm -rf .git
    npm install
    bower install

Development
===========

    gulp
